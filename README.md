## Descripción

Frontend de una aplicación para llevar el seguimiento de pacientes para veterinarios.

## Instalación

```bash
$ npm install
```

## Ejecutando la aplicación

```bash
# desarrollo
$ npm run dev
# producción
$ npm run build
```
